Internal API Reference

These functions are not exported but may be useful for advanced usage or are documented here for contributor reference.

MDH Parsing

MapVBVD.compute_dims! — Function
compute_dims!(s::ScanData)

Compute dimension sizes from acquisition metadata.

MapVBVD.readMDH! — Function
readMDH!(s::ScanData, mdh::MDH, filePos, useScan)

Extract MDH values for the selected scans and populate the ScanData.

MapVBVD.tryAndFixLastMdh! — Function
tryAndFixLastMdh!(s::ScanData)

Attempt to recover from read errors by trimming the last acquisition.

MapVBVD.calcIndices — Function
calcIndices(s::ScanData) -> (ixToRaw, ixToTarget)

Calculate mapping from MDH acquisitions to target array positions.

MapVBVD.calcRange — Function
calcRange(s::ScanData, S) -> (selRange, selRangeSz, outSize)

Calculate selection ranges for data retrieval.

MapVBVD.readData — Function
readData(s, mem; cIxToTarg, cIxToRaw, selRange, selRangeSz, outSize)

Read raw data from file. Core I/O routine.

Data Helpers

MapVBVD.fixLoopCounter! — Function
fixLoopCounter!(s, loop, newLoop)

Replace a loop counter with new values. Note: because AcquisitionMeta is immutable, this rebuilds the entire meta struct.

MapVBVD.average_dim — Function
average_dim(s::ScanData) -> Vector{Bool}

Return a length-16 boolean vector indicating which dimensions are averaged.

MapVBVD.loop_counter — Function
loop_counter(meta::AcquisitionMeta, dim_index::Int) -> Vector{Int32}

Get the loop counter vector for a given dimension index (DIMLIN..DIMIDE).

MapVBVD.dim_extent — Function
dim_extent(dims::DimSizes, i::Int) -> Int

Get the extent for dimension i (1-based, matching DIM_* constants).

Header Parsing

MapVBVD.parse_ascconv — Function
parse_ascconv(buffer::AbstractString) -> NestedDict

Parse ASCCONV section from Siemens header buffer into a tree structure. E.g., sKSpace.lBaseResolution = 256 becomes accessible as result.sKSpace.lBaseResolution.

MapVBVD.parse_xprot — Function
parse_xprot(buffer::AbstractString) -> Dict

Parse XProtocol section from Siemens header buffer.

MapVBVD.parse_buffer — Function
parse_buffer(buffer::AbstractString) -> NestedDict

Parse a complete header buffer (both ASCCONV and XProtocol sections) into a single NestedDict tree.

Utilities

MapVBVD.cumtrapz — Function
cumtrapz(y)

Cumulative trapezoidal integration (like scipy.integrate.cumulative_trapezoid). Returns a vector of length(y) with 0 prepended.
